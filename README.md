
# 📈 Stock Trading Platform

A Java-based Stock Trading Platform developed using Object-Oriented Programming (OOP) principles. This project simulates a basic stock market environment where users can view available stocks, buy and sell shares, manage their portfolio, and track transaction history.

## 🚀 Features

- 📊 Display available stocks and their current prices
- 💰 Buy and sell stocks
- 👤 User portfolio management
- 📜 Transaction history tracking
- 💵 Balance management
- 🧩 Object-Oriented Design using Java classes
- 🗄️ MySQL database integration (optional/implemented)
- 🖥️ Console-based interface (GUI can be added in future)

## 🛠️ Technologies Used

- Object-Oriented Programming (OOP)
- MySQL

## ⚙️ How It Works

1. The application loads available stocks.
2. The user can view the stock market.
3. Users can buy or sell stocks based on their account balance.
4. Purchased stocks are stored in the user's portfolio.
5. Every trade is recorded as a transaction.
6. Portfolio and transaction history can be viewed at any time.
